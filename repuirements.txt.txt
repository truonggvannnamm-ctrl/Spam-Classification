pandas → xử lý dữ liệu dạng DataFrame

numpy → tính toán số học, mảng

re → xử lý biểu thức chính quy (regex)

matplotlib.pyplot → vẽ biểu đồ cơ bản

wordcloud.WordCloud → tạo word cloud

seaborn → trực quan hóa nâng cao, heatmap

sklearn.model_selection.train_test_split → chia dữ liệu train/test

sklearn.feature_extraction.text.TfidfVectorizer → TF-IDF vectorizer

sklearn.naive_bayes.MultinomialNB → mô hình Naive Bayes

sklearn.linear_model.LogisticRegression → mô hình Logistic Regression

sklearn.svm.LinearSVC → mô hình Linear SVM

sklearn.metrics:

accuracy_score → độ chính xác

precision_score → precision

recall_score → recall

f1_score → F1-score

confusion_matrix → ma trận nhầm lẫn

classification_report → báo cáo chi tiết

nltk:

nltk.corpus.stopwords → stopwords tiếng Anh

nltk.stem.WordNetLemmatizer → lemmatization

nltk.download('stopwords') → tải stopwords

nltk.download('wordnet') → tải WordNet cho lemmatization
